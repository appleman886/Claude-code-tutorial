第一步：安装 Node.js

Node.js 是 Claude Code 的运行环境，必须先安装。

🪟 Windows 一键安装

步骤 1：打开命令提示符

  • 按下 Win + R 键
  • 输入 cmd 并回车
  • 或者按下 Win + X，选择"命令提示符"或"Windows PowerShell"

步骤 2：复制并运行安装命令

点击下方代码块复制命令，然后在命令提示符窗口中右键点击粘贴（会自动执行）：

winget install OpenJS.NodeJS.LTS

等待安装完成，期间可能需要你按 Y 确认。

步骤 3：验证安装

关闭当前命令提示符窗口，重新打开，输入以下命令：

node -v

如果显示版本号（例如 v20.x.x），说明安装成功！

💡 如果 winget 不可用

如果提示 winget 命令不存在，请访问 Node.js 官网 下载安装包，双击安装即可。

🍎 macOS 一键安装

步骤 1：打开终端

  • 按下 Command + 空格 打开 Spotlight 搜索
  • 输入"终端"或"Terminal"
  • 按回车打开终端应用
  • 或者在 Finder 中打开"应用程序" → "实用工具" → "终端"

步骤 2：检查是否安装了 Homebrew

在终端中输入以下命令，按回车：

brew --version

如果显示版本号，说明已安装。如果提示"command not found"，需要先安装 Homebrew。

步骤 3：安装 Homebrew（如果需要）

点击下方代码块复制命令，然后在终端窗口中粘贴（Command + V），按回车执行：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装过程中需要你输入管理员密码（输入时不会显示字符），按回车确认。

步骤 4：使用 Homebrew 安装 Node.js

在终端中输入以下命令：

brew install node

等待安装完成。

步骤 5：验证安装

在终端中输入以下命令：

node -v

如果显示版本号，说明安装成功！

🐧 Linux 一键安装

步骤 1：打开终端

  • 按下 Ctrl + Alt + T 快捷键打开终端
  • 或者在应用菜单中搜索"Terminal"
  • 右键点击桌面或文件夹空白处，选择"在终端中打开"（不同发行版可能略有不同）

步骤 2：确认你的 Linux 发行版

在终端中输入以下命令查看系统版本：

cat /etc/os-release

查看输出中的 NAME 或 PRETTY_NAME 来确认你的发行版。

步骤 3：根据发行版选择安装命令

Ubuntu / Debian / Kali Linux:

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs

CentOS / RHEL / Fedora (RPM 系列):

curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo yum install -y nodejs

注意：Fedora 可以使用 dnf 替代 yum

Arch Linux / Manjaro:

sudo pacman -S nodejs npm

Fedora:

sudo dnf install nodejs

openSUSE:

sudo zypper install nodejs

如何粘贴命令到终端：

  • 复制命令： 选中代码块中的命令，按 Ctrl + C 复制
  • 粘贴命令： 在终端中按 Ctrl + Shift + V 粘贴（大部分 Linux 终端）
  • 或者： 在终端中右键点击，选择"粘贴"
  • 或者： 有些终端支持 Shift + Insert 粘贴

步骤 4：安装过程中的提示

  • 如果提示输入密码，输入你的用户密码（输入时不会显示字符），按回车
  • 如果询问是否继续安装，输入 y 然后按回车
  • 等待安装完成，可能需要几分钟

步骤 5：验证安装

在终端中输入以下命令：

node -v

如果显示版本号（例如 v20.x.x），说明安装成功！

💡 常见问题

  • 提示权限不足： 在命令前加 sudo
  • curl 命令不存在： 先运行 sudo apt install curl 或 sudo yum install curl
  • 找不到对应发行版的命令： 访问 Node.js 官网 查看详细安装指南

Git 是版本控制工具，Claude Code 的一些功能会用到。

🪟 Windows 一键安装

步骤 1：打开命令提示符

  • 按下 Win + R 键
  • 输入 cmd 并回车

步骤 2：复制并运行安装命令

点击下方代码块复制命令，然后在命令提示符窗口中右键点击粘贴：

等待安装完成。

步骤 3：验证安装

关闭当前命令提示符窗口，重新打开，输入以下命令：

如果显示版本号（例如 git version 2.x.x），说明安装成功！

🍎 macOS 一键安装

步骤 1：打开终端

  • 按下 Command + 空格 打开 Spotlight 搜索
  • 输入"终端"或"Terminal"
  • 按回车打开终端应用

步骤 2：选择安装方式

方法一：使用 Homebrew（推荐）

在终端中输入以下命令：

brew install git

方法二：使用 Xcode 命令行工具

在终端中输入以下命令：

xcode-select --install

系统会弹出对话框，点击"安装"按钮，系统会自动安装 Git 和其他开发工具。

步骤 3：验证安装

在终端中输入以下命令：

第三步：安装 Claude Code

现在可以安装 Claude Code 命令行工具了。

1. 安装 Claude Code

Windows 用户：打开命令提示符（CMD）或 PowerShell

macOS/Linux 用户：打开终端

输入以下命令：

npm install -g @anthropic-ai/claude-code

等待安装完成，可能需要几分钟时间。

2. 验证安装

安装完成后，输入以下命令查看版本：

claude --version

如果显示版本号，说明安装成功！

3. 初次使用

输入以下命令启动 Claude Code：

claude

首次使用时，系统会引导您进行配置。请选择跳过 API 密钥配置，我们将在下一步配置智谱 AI。

💡 小贴士

如果安装过程中遇到权限问题，请以管理员身份运行命令提示符。请使用 sudo 命令或在命令前加 sudo。

第四步：配置智谱 AI API

最后一步是配置智谱 AI 的 API，让 Claude Code 使用智谱的模型服务。

1. 注册智谱 AI 账号

访问智谱开放平台，注册或登录账号。

访问智谱开放平台

2. 获取 API Key

登录后，按照以下步骤获取 API Key：

  • 进入个人中心
  • 点击"项目管理"
  • 点击"API keys"
  • 创建新的 API Key
  • 复制生成的 API Key（妥善保管！）

3. 配置环境变量（Windows）

方法一：使用 CMD（推荐）

打开命令提示符，设置 API Key：

setx ANTHROPIC_AUTH_TOKEN "你的智谱API密钥"

设置 API 地址：

setx ANTHROPIC_BASE_URL "https://open.bigmodel.cn/api/paas/v4"

方法二：使用 PowerShell

打开 PowerShell，设置 API Key：

$env:ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"
[Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", $env:ANTHROPIC_AUTH_TOKEN, "User")

设置 API 地址：

$env:ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", $env:ANTHROPIC_BASE_URL, "User")

注意：需要重启命令行窗口才能生效。

3. 配置环境变量（macOS）

方法一：临时配置（当前会话有效）

打开终端，设置 API Key：

export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"

设置 API 地址：

export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"

方法二：永久配置（推荐）

打开终端，编辑 shell 配置文件（zsh 或 bash）：

echo 'export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"' >> ~/.zshrc
echo 'export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"' >> ~/.zshrc
source ~/.zshrc

如果使用 bash，将 ~/.zshrc 改为 ~/.bash_profile

3. 配置环境变量（Linux）

方法一：临时配置（当前会话有效）

打开终端，设置 API Key：

export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"

设置 API 地址：

export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"

方法二：永久配置（推荐）

打开终端，编辑 shell 配置文件：

echo 'export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"' >> ~/.bashrc
echo 'export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"' >> ~/.bashrc
source ~/.bashrc

如果使用 zsh，将 ~/.bashrc 改为 ~/.zshrc

4. 验证配置

打开新的命令行窗口，启动 Claude Code：

claude

如果配置正确，Claude Code 将使用智谱 AI 的模型服务！

您已经成功安装和配置了 Claude Code，现在可以开始使用了！
