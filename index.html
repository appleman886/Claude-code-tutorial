快速开始

  1. 安装 Node.js 运行环境
  2. 安装 Git 版本控制工具
  3. 安装 Claude Code 命令行工具
  4. 配置智谱 AI API 密钥
  5. 开始使用 Claude Code

第一步：安装 Node.js

Node.js 是 Claude Code 的运行环境，必须先安装。

1. 下载 Node.js

访问 Node.js 官方网站下载 LTS 版本（长期支持版本）。

访问 Node.js 官网

2. 运行安装程序

双击下载的 .msi 文件，按照提示进行安装：

  • ✓ 选择默认安装路径
  • ✓ 确保勾选 "Add to PATH"（自动添加到环境变量）
  • ✓ 完成安装后重启电脑或命令行窗口

3. 验证安装

打开命令提示符（CMD）或 PowerShell，输入以下命令：

node -v

如果显示版本号（例如 v20.x.x），说明安装成功！

第二步：安装 Git

Git 是版本控制工具，Claude Code 的一些功能会用到。

1. 下载 Git

访问 Git 官方网站下载 Windows 版本。

下载 Git for Windows

2. 运行安装程序

运行下载的安装程序，主要配置项：

  • ✓ 选择默认编辑器（建议 VS Code）
  • ✓ "PATH environment" 选择 "Git from the command line"
  • ✓ "HTTPS transport" 选择默认
  • ✓ 其他选项保持默认即可

3. 验证安装

打开命令提示符（CMD）或 PowerShell，输入以下命令：

git --version

如果显示版本号（例如 git version 2.x.x），说明安装成功！

第三步：安装 Claude Code

现在可以安装 Claude Code 命令行工具了。

1. 安装 Claude Code

打开命令提示符（CMD）或 PowerShell，输入以下命令：

npm install -g @anthropic-ai/claude-code

等待安装完成，可能需要几分钟时间。

2. 验证安装

安装完成后，输入以下命令查看版本：

claude --version

如果显示版本号，说明安装成功！

3. 初次使用

输入以下命令启动 Claude Code：

claude

首次使用时，系统会引导您进行配置。请选择跳过 API 密钥配置，我们将在下一步配置智谱 AI。

💡 小贴士

如果安装过程中遇到权限问题，请以管理员身份运行命令提示符。

第四步：配置智谱 AI API

最后一步是配置智谱 AI 的 API，让 Claude Code 使用智谱的模型服务。

1. 注册智谱 AI 账号

访问智谱开放平台，注册或登录账号。

访问智谱开放平台

2. 获取 API Key

登录后，按照以下步骤获取 API Key：

  • 进入个人中心
  • 点击"项目管理"
  • 点击"API keys"
  • 创建新的 API Key
  • 复制生成的 API Key（妥善保管！）

3. 配置环境变量

在 Windows 系统中设置环境变量：

setx ANTHROPIC_AUTH_TOKEN "你的智谱API密钥"

或者使用 PowerShell：

$env:ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"

注意：需要重启命令行窗口才能生效。

4. 配置 API 地址

设置智谱 AI 的 API 地址：

setx ANTHROPIC_BASE_URL "https://open.bigmodel.cn/api/paas/v4"

或使用 PowerShell：

$env:ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"

5. 验证配置

打开新的命令行窗口，启动 Claude Code：

claude

如果配置正确，Claude Code 将使用智谱 AI 的模型服务！

🎉 恭喜！

您已经成功安装和配置了 Claude Code，现在可以开始使用了！
