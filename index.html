第一步：安装 Node.js

Node.js 是 Claude Code 的运行环境，必须先安装。

🪟 Windows 一键安装

打开命令提示符（CMD）或 PowerShell，输入以下命令：

winget install OpenJS.NodeJS.LTS

如果没有 winget，请访问 Node.js 官网 下载安装。

验证安装

重启命令行窗口，输入以下命令：

node -v

如果显示版本号（例如 v20.x.x），说明安装成功！

🍎 macOS 一键安装

方法一：使用 Homebrew（推荐）

brew install node

如果没有 Homebrew，先安装 Homebrew：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

方法二：使用安装包

访问 Node.js 官网下载

验证安装

打开终端，输入以下命令：

node -v

如果显示版本号，说明安装成功！

🐧 Linux 一键安装

Ubuntu/Debian:

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs

CentOS/RHEL:

curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo yum install -y nodejs

Arch Linux:

sudo pacman -S nodejs npm

Fedora:

sudo dnf install nodejs

验证安装

打开终端，输入以下命令：

node -v

如果显示版本号，说明安装成功！

第二步：安装 Git

Git 是版本控制工具，Claude Code 的一些功能会用到。

🪟 Windows 一键安装

打开命令提示符（CMD）或 PowerShell，输入以下命令：

winget install Git.Git

如果没有 winget，请访问 Git 官网 下载安装。

验证安装

重启命令行窗口，输入以下命令：

git --version

如果显示版本号（例如 git version 2.x.x），说明安装成功！

🍎 macOS 一键安装

方法一：使用 Homebrew（推荐）

brew install git

方法二：使用 Xcode 命令行工具

xcode-select --install

系统会自动安装 Git 和其他开发工具。

方法三：使用安装包

访问 Git 官网下载

验证安装

打开终端，输入以下命令：

git --version

如果显示版本号，说明安装成功！

🐧 Linux 一键安装

Ubuntu/Debian:

sudo apt update && sudo apt install -y git

CentOS/RHEL:

sudo yum install -y git

Arch Linux:

sudo pacman -S git

Fedora:

sudo dnf install -y git

openSUSE:

sudo zypper install -y git

验证安装

打开终端，输入以下命令：

git --version

如果显示版本号，说明安装成功！

第三步：安装 Claude Code

现在可以安装 Claude Code 命令行工具了。

1. 安装 Claude Code

打开命令行工具（CMD/PowerShell/终端），输入以下命令：

npm install -g @anthropic-ai/claude-code

等待安装完成，可能需要几分钟时间。

2. 验证安装

安装完成后，输入以下命令查看版本：

claude --version

如果显示版本号，说明安装成功！

3. 初次使用

输入以下命令启动 Claude Code：

claude

首次使用时，系统会引导您进行配置。请选择跳过 API 密钥配置，我们将在下一步配置智谱 AI。

💡 小贴士

如果安装过程中遇到权限问题，请以管理员身份运行命令提示符。请使用 sudo 命令或在命令前加 sudo。

第四步：配置智谱 AI API

最后一步是配置智谱 AI 的 API，让 Claude Code 使用智谱的模型服务。

1. 注册智谱 AI 账号

访问智谱开放平台，注册或登录账号。

访问智谱开放平台

2. 获取 API Key

登录后，按照以下步骤获取 API Key：

  • 进入个人中心
  • 点击"项目管理"
  • 点击"API keys"
  • 创建新的 API Key
  • 复制生成的 API Key（妥善保管！）

3. 配置环境变量（Windows）

方法一：使用 CMD（推荐）

设置 API Key：

setx ANTHROPIC_AUTH_TOKEN "你的智谱API密钥"

设置 API 地址：

setx ANTHROPIC_BASE_URL "https://open.bigmodel.cn/api/paas/v4"

方法二：使用 PowerShell

设置 API Key：

$env:ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"
[Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", $env:ANTHROPIC_AUTH_TOKEN, "User")

设置 API 地址：

$env:ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", $env:ANTHROPIC_BASE_URL, "User")

注意：需要重启命令行窗口才能生效。

3. 配置环境变量（macOS）

方法一：临时配置（当前会话有效）

设置 API Key：

export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"

设置 API 地址：

export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"

方法二：永久配置（推荐）

编辑 shell 配置文件（zsh 或 bash）：

echo 'export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"' >> ~/.zshrc
echo 'export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"' >> ~/.zshrc
source ~/.zshrc

如果使用 bash，将 ~/.zshrc 改为 ~/.bash_profile

3. 配置环境变量（Linux）

方法一：临时配置（当前会话有效）

设置 API Key：

export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"

设置 API 地址：

export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"

方法二：永久配置（推荐）

编辑 shell 配置文件：

echo 'export ANTHROPIC_AUTH_TOKEN="你的智谱API密钥"' >> ~/.bashrc
echo 'export ANTHROPIC_BASE_URL="https://open.bigmodel.cn/api/paas/v4"' >> ~/.bashrc
source ~/.bashrc

如果使用 zsh，将 ~/.bashrc 改为 ~/.zshrc

4. 验证配置

打开新的命令行窗口，启动 Claude Code：

claude

如果配置正确，Claude Code 将使用智谱 AI 的模型服务！
