什么是 Claude Code？

Claude Code 是 Anthropic 公司推出的 AI 编程助手，通过命令行界面与 Claude 人工智能模型交互。 它可以帮助你编写代码、调试程序、解释复杂概念、重构代码，是一个强大的开发工具。

为什么要安装 Claude Code？

  • 🎯 提升效率 - 自动生成代码，减少重复劳动
  • 🔍 智能调试 - 快速定位和修复 bug
  • 📚 代码解释 - 理解复杂的代码逻辑
  • ✨ 代码优化 - 重构和改进代码质量
  • 🌍 多语言支持 - 支持 Python、JavaScript、C++ 等主流语言

下载安装脚本

选择您的操作系统，下载对应的一键安装脚本。

🪟 Windows 版本

🚀

完全一键安装包 (推荐)

自动安装 Node.js + Git + Claude Code，最简单！

下载 install-all.bat

下载后右键 → "以管理员身份运行"

或者分别安装：

1. Node.js 安装脚本 (带错误处理) 下载 install-nodejs.bat
2. Git 安装脚本 (带错误处理) 下载 install-git.bat
3. Claude Code 安装脚本 (带错误处理) 下载 install-claude.bat

💡 使用说明

  1. 点击上方链接下载脚本文件（.bat 文件）
  2. 右键点击脚本，选择"以管理员身份运行"
  3. 脚本会自动检测和处理权限问题
  4. 如果出错，脚本会提供详细的解决方法
  5. 安装完成后，关闭窗口并重新打开命令行
  6. 运行 claude 启动程序

⚠️ 常见问题

  • 提示权限不足：右键脚本 → "以管理员身份运行"
  • PowerShell 策略限制：脚本会自动提示并帮助修改
  • 下载失败：脚本会使用国内镜像，无需代理
  • 安装后命令无效：重启电脑后重新打开命令行

配置 API 密钥

安装完成后，需要配置 API 密钥才能使用 Claude Code。

🚀 快速配置（推荐）

使用智谱官方助手，交互式向导自动配置：

npx @z_ai/coding-helper

说明：首次运行会自动下载安装，按照提示操作即可。

选择您的操作系统：

方案一：智谱 AI

步骤 1：获取 API Key

  1. 访问 智谱开放平台
  2. 注册/登录账号
  3. 进入控制台 → API-KEY 管理
  4. 创建 API Key 并复制保存

步骤 2：配置环境变量

打开命令提示符，运行以下命令：

setx ANTHROPIC_AUTH_TOKEN "你的智谱 API Key"
setx ANTHROPIC_BASE_URL "https://open.bigmodel.cn/api/paas/v4"

步骤 3：重启命令行

关闭当前窗口，重新打开命令提示符，运行 claude 启动程序。

步骤 1：获取 API Key

  1. 访问 智谱开放平台
  2. 注册/登录账号
